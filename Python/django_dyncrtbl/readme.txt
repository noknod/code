Задание:

Разработать приложение, позволяющее динамически создавать таблицы через форму. 

1) В форме заполняются поля, соответствующие названию таблицы, а также названию
 и типу поля (для начала только IntegerField, CharField и TextField). 

2) После заполнения формы создаётся таблица БД, регистрируется через 
contenttypes_framework и site.admin.register. 

3) Ручного рестарта приложения требоваться не должно.

- - -

Среда разработки:

Python 3.4.3, Django 1.8, SQLite3
